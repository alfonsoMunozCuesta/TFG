\section*{Design and implementation of an application for staff management in the hospitality sector}

\noindent
Merichal Granados, Antonio(2025). Trabajo de Fin de Grado. Córdoba: Universidad de Córdoba.

\section*{Abstract}
    This Final Degree Project aims to develop an application for managing employee shifts in the hospitality sector, particularly for events such as weddings and celebrations with catering services. The proposed solution enables multi-company management, with clearly differentiated roles for administrators and employees, as well as functionalities for event planning, staff assignment based on configurable criteria, and internal communication through notifications. Additionally, it centralizes personnel information, facilitates payment tracking, and generates customized reports.


\section*{Keywords}
Personnel management, event planning, web application.
