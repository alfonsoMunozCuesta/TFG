
\section*{Diseño e implementación de una aplicación para la gestión de personal en el sector hostelero}

\noindent
Merichal Granados, Antonio(2025). Trabajo de Fin de Grado. Córdoba: Universidad de Córdoba.

\section*{Resumen}
    Este Trabajo Fin de Grado tiene como objetivo el desarrollo de una aplicación para la gestión de turnos de empleados en elsector hostelero, especialmente en eventos como bodas y celebraciones con servicios de catering. La solución propuesta permite la gestión multicompañía, con roles diferenciados para administradores y trabajadores, y funcionalidades para la planificación de eventos, asignación de personal según criterios configurables, y comunicación interna mediante notificaciones. Además, centraliza la información del personal, facilita el seguimiento de pagos y genera informes personalizados.

\section*{Palabras clave}
Gestion personal, planificación eventos, aplicación web.
