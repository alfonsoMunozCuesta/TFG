Coloca en esta carpeta el modelo GGUF utilizado por el servidor local de IA.

Nombre esperado por defecto:
qwen2.5-1.5b-instruct-q4_k_m.gguf

Enlace de descarga:
https://drive.google.com/file/d/1G9zDMnDXh1nBYblO3OPhTiROFFuCHuGS/view?usp=drive_link

El archivo del modelo no se incluye en el repositorio porque puede ocupar mucho
espacio. Si se utiliza otro modelo o una ruta diferente, hay que ajustar la
variable MODEL_PATH antes de ejecutar START_LLAMA_SERVER.bat.
